# AI-Powered LinkedIn Content Automation

**Overview:** 
Built a full-stack web application that automatically generates, formats, and publishes LinkedIn posts. The system uses AI to write content, but includes a mandatory human review step before anything is posted online.

**Technologies Used:** Python, FastAPI, React, Node.js, Large Language Models (LLM / Groq API), LinkedIn REST API, Git.

**Key Responsibilities & Achievements:**

* **Full-Stack Development:** Designed and built both the frontend (React) and the backend (Python/FastAPI) from scratch to create a seamless user experience.
* **AI Integration:** Connected the backend to a Large Language Model (LLM) to automatically generate engaging post ideas, structured content (with bullet points), and hashtags.
* **Human-in-the-Loop Workflow:** Engineered a multi-step approval process where users can review, edit, and approve the AI-generated content before it goes live, ensuring high quality and brand safety.
* **API Development & Integration:** Successfully integrated the official LinkedIn REST API to handle automated publishing directly to user profiles using secure OAuth 2.0 authentication.
* **UI/UX Implementation:** Built a responsive, easy-to-use React dashboard that provides real-time error handling, input validation, and clear feedback to the user.
* **Prompt Engineering:** Designed strict AI instructions (prompts) to ensure output is always professionally formatted, easily readable, and optimized for social media engagement.

**Impact:** 
Created a complete, production-ready tool that drastically reduces the manual time needed to create and share professional social media content, demonstrating strong practical skills in both modern web development and applied AI.
